# Android FFmpeg入门项目
这个项目可以让你了解以下内容：
1. 如何编译多种CPU指令集的FFmpeg动态库
2. 如何配置CMakeLists.txt和build.gradle构建Android项目
# 编译环境
- ndk-r16
- ffmpeg-4.0.2
